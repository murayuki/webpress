# 黑色沙漠中文化工具 - 安裝說明

# 安裝前須知

安裝後遊戲更新點擊還原會將之前安裝時的英文備份還原。如無所謂可以使用 (PS: 強制下載也一樣)

或是手動將 遊戲目錄中 “ads” 裡面的備份刪除 副檔名.bak 刪除在點擊安裝即可，注意(請選擇 下載類型) 之前安裝過字型可已不用再次安裝。

# 重要須知

本程式有將所有的DLL封裝，VirusTotal檢測有一項病毒我自己是爬過文章那一項是誤報

請自行斟酌是否要使用本程式，本程式下載連結只有在本教學頁面，其他地方下載一概不理。

# 步驟 - 1

# 兩種的方式可以執行

  1. 將程式放置 遊戲目錄中 後直接啟動程式
  2. 直接啟動程式，選擇 "是" 選擇 遊戲目錄

範例路徑: (PS: "X" 代表你自己的硬碟位置) X:\Black Desert Online X:\Steam\steamapps\common\Black Desert Online

# 步驟 - 2

# 正常開啟

選擇替換的字體 "英文" PS: 該功能如果使用 英文 以外的語言需要啟動遊戲 > 遊戲設定中設定

選擇安裝的下載方式 第一次安裝 請選擇 語言文件與字體 接著點擊 安裝 等待安裝完成即可

  • 還原功能

    還原功能 是將原本安裝的的語言文件與字體刪除 PS: 完全刪除功能預設打勾 (不打勾不刪除字體僅刪除語言文件)

  • 強制下載功能

    使用強制下載 建議基於無法正常下載才使用該功能，該功能會將備份檔案直接刪除如遊戲檔案有問題請檢查更新或是重新安裝

遊戲更新重新安裝方式

因遊戲更新會將中文語言檔案覆蓋 請使用程式 左上角 功能表中 "遊戲更新修正" 完成後直接點擊 安裝 即可 (僅下載 語言文件)
